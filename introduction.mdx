# Welcome to Product Graph API

The Product Graph API provides a comprehensive set of tools for accessing, enriching, and managing product data across multiple languages. With powerful search capabilities, multilingual support, and advanced analytics, our API helps you build better e-commerce experiences.

## Why Product Graph?

Product data is often fragmented, inconsistent, and locked in language silos. Product Graph solves these challenges by providing:

* **Unified Product Information** - Access comprehensive product data from a single source

* **Multilingual Support** - Get product information in multiple languages with one API call

* **Advanced Search** - Find products using names, GTINs, URLs, or a combination of parameters

* **Historical Tracking** - Monitor how product data changes over time

* **Data Enrichment** - Enhance product listings with high-quality descriptions, attributes, and media

## Quick Authentication Guide

```bash
# All API requests require an API key in the header
curl -X GET "https://api.productgraph.org/v1/credits" \
  -H "X-API-Key: your-api-key"
```

<Note>
  Your API key is available in the [Product Graph Dashboard](https://dashboard.productgraph.org).
  Keep it secure and never expose it in client-side code!
</Note>

## Example: Search for Products

```javascript
// Search for Sony wireless headphones in English and German
const searchData = {
  query: "wireless headphones",
  languages: ["en", "de"],
  filter: {
    brand: "Sony"
  }
};

fetch("https://api.productgraph.org/v1/search", {
  method: "POST",
  headers: {
    "X-API-Key": apiKey,
    "Content-Type": "application/json"
  },
  body: JSON.stringify(searchData)
})
.then(response => response.json())
.then(data => {
  console.log("Search job initiated:", data);
  // The search is asynchronous, check results using the job_id
})
.catch(error => console.error("Error:", error));
```

## Asynchronous Processing

Many Product Graph API operations process asynchronously for efficiency:

```javascript
// Check the status of a search job
fetch(`https://api.productgraph.org/v1/search/${jobId}`, {
  headers: {
    "X-API-Key": apiKey
  }
})
.then(response => response.json())
.then(data => {
  if (data.status === "completed") {
    console.log("Search results:", data.results);
  } else {
    console.log("Job status:", data.status);
  }
})
```

For production applications, use [webhooks](/product-graph/webhooks) to receive notifications when jobs complete.
