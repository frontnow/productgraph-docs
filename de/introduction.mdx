# Willkommen bei der Product Graph API

Die Product Graph API bietet ein umfassendes Set an Werkzeugen für den Zugriff, die Anreicherung und die Verwaltung von Produktdaten in mehreren Sprachen. Mit leistungsstarken Suchfunktionen, mehrsprachiger Unterstützung und fortschrittlicher Analytik hilft unsere API Ihnen, bessere E-Commerce-Erfahrungen zu schaffen.

## Warum Product Graph?

Produktdaten sind oft fragmentiert, inkonsistent und in Sprachsilos eingeschlossen. Product Graph löst diese Herausforderungen durch folgende Funktionen:

* **Vereinheitlichte Produktinformationen** - Zugriff auf umfassende Produktdaten aus einer einzigen Quelle

* **Mehrsprachige Unterstützung** - Erhalten Sie Produktinformationen in mehreren Sprachen mit einem einzigen API-Aufruf

* **Erweiterte Suche** - Finden Sie Produkte anhand von Namen, GTINs, URLs oder einer Kombination von Parametern

* **Historisches Tracking** - Überwachen Sie, wie sich Produktdaten im Laufe der Zeit ändern

* **Datenanreicherung** - Verbessern Sie Produktlistings mit hochwertigen Beschreibungen, Attributen und Medien

## Schnellguide zur Authentifizierung

```bash
# Alle API-Anfragen erfordern einen API-Schlüssel im Header
curl -X GET "https://api.productgraph.org/v1/credits" \
  -H "X-API-Key: your-api-key"
```

<Note>
  Ihr API-Schlüssel ist im [Product Graph Dashboard](https://dashboard.productgraph.org) verfügbar.
  Halten Sie ihn sicher und geben Sie ihn niemals in clientseitigem Code preis!
</Note>

## Beispiel: Nach Produkten suchen

```javascript
// Suche nach Sony-Funkkopfhörern auf Englisch und Deutsch
const searchData = {
  query: "wireless headphones",
  languages: ["en", "de"],
  filter: {
    brand: "Sony"
  }
};

fetch("https://api.productgraph.org/v1/search", {
  method: "POST",
  headers: {
    "X-API-Key": apiKey,
    "Content-Type": "application/json"
  },
  body: JSON.stringify(searchData)
})
.then(response => response.json())
.then(data => {
  console.log("Suchauftrag initiiert:", data);
  // Die Suche ist asynchron, überprüfen Sie die Ergebnisse mit der job_id
})
.catch(error => console.error("Fehler:", error));
```

## Asynchrone Verarbeitung

Viele Product Graph API-Operationen werden aus Effizienzgründen asynchron verarbeitet:

```javascript
// Überprüfen des Status eines Suchauftrags
fetch(`https://api.productgraph.org/v1/search/${jobId}`, {
  headers: {
    "X-API-Key": apiKey
  }
})
.then(response => response.json())
.then(data => {
  if (data.status === "completed") {
    console.log("Suchergebnisse:", data.results);
  } else {
    console.log("Auftragsstatus:", data.status);
  }
})
```

Für Produktionsanwendungen verwenden Sie [Webhooks](/de/product-graph/webhooks), um Benachrichtigungen zu erhalten, wenn Aufträge abgeschlossen sind.
