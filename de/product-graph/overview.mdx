# Product Graph API Überblick

ProductGraph.org bietet einen umfassenden Service zur Anreicherung und Verwaltung von Produktdaten, der entwickelt wurde, um die Qualität von Produktdaten in mehreren Sprachen zu verbessern. Unsere API ermöglicht es Unternehmen, auf detaillierte Produktinformationen zuzugreifen, Änderungen im Laufe der Zeit zu verfolgen und Produktinhalte effizient zu verwalten.

<Note>
Die Product Graph API ist RESTful und liefert Antworten im JSON-Format. Alle API-Endpunkte sind unter `https://api.productgraph.org/v1/` verfügbar.
</Note>

## Was diese API bietet

<Accordion title="Produktsuchmöglichkeiten">
  Unsere Suchmaschine ermöglicht es Ihnen, Produkte mit verschiedenen Identifikatoren zu finden:
  - Produktnamen mit unscharfer Übereinstimmung und Relevanzbewertung
  - Globale Handelsartikelnummern (GTINs) einschließlich UPC, EAN, ISBN
  - Produkt-URLs von unterstützten Händlern und Herstellern
  - Filter für Marken, Kategorien, Attribute und mehr
</Accordion>

<Accordion title="Detaillierte Datenanreicherung">
  Verbessern Sie Ihre Produktdaten mit:
  - Umfassenden Produktbeschreibungen und Funktionen
  - Strukturierten technischen Spezifikationen und Attributen
  - Hochwertigen Produktbildern mit KI-gestützter Analyse
  - Produktvideos und Werbeinhalten
  - Benutzerbewertungen und Ratings
  - Zugehörige Produktdokumentation (Handbücher, Anleitungen)
  - SEO-Optimierungsvorschläge
</Accordion>

<Accordion title="Mehrsprachige Unterstützung">
  Zugriff auf Produktdaten in mehreren Sprachen:
  - Automatische Übersetzung von Produktinformationen
  - Sprachspezifische Inhaltsverwaltung
  - Sprachübergreifende Suchfunktionen
  - Sprachqualitätsbewertung und Verbesserungsvorschläge
</Accordion>

<Accordion title="Historische Datenverfolgung">
  Überwachen Sie, wie sich Produktdaten im Laufe der Zeit ändern:
  - Versionsverlauf für alle Produktattribute
  - Änderungsprotokolle mit detaillierten Modifikationen
  - Zeitstempelbasierter Datenabruf
  - Trendanalyse für Produktattribute
</Accordion>

<Accordion title="Technische Funktionen">
  Unsere API umfasst:
  - Asynchrone Verarbeitung für umfangreiche Operationen
  - Auftragsstatus-Verfolgung für langwierige Aufgaben
  - Webhook-Integration für Echtzeit-Benachrichtigungen
  - Detaillierte Analytik und Nutzungsberichte
  - Datenqualitätsbewertung mit Verbesserungsvorschlägen
</Accordion>

## Was diese API NICHT bietet

<Warning>
Die folgenden Funktionen liegen außerhalb des Umfangs unseres aktuellen API-Angebots. Bitte überprüfen Sie unsere Roadmap oder kontaktieren Sie uns für Informationen über zukünftige Pläne.
</Warning>

<Tabs>
  <Tab title="Wettbewerbsanalyse">
    Unsere API bietet keine Wettbewerbsinformationen oder Marktpositionierungsdaten. Sie konzentriert sich auf Produktinformationen und nicht auf Marktanalysen.
  </Tab>
  <Tab title="Preisdaten">
    Wir verfolgen keine Preis- oder Verfügbarkeitsinformationen. Unsere API konzentriert sich auf Produktattribute und -inhalte und nicht auf kommerzielle Bedingungen.
  </Tab>
  <Tab title="Prognosen">
    Bedarfsprognosen, Trendanalysen und prädiktive Funktionen sind nicht Teil unseres aktuellen Angebots. Wir konzentrieren uns auf historische und aktuelle Produktdaten.
  </Tab>
  <Tab title="Compliance-Daten">
    Obwohl wir Produktspezifikationen bereitstellen, bieten wir keine spezifische Überprüfung der Nachhaltigkeits- oder regulatorischen Compliance an.
  </Tab>
  <Tab title="Bestand">
    Unsere API behandelt keine Bestandsverwaltung oder Echtzeitlagerbestände. Sie konzentriert sich auf Produktdaten und nicht auf Bestand.
  </Tab>
  <Tab title="Empfehlungen">
    Produktempfehlungssysteme oder Personalisierungsfunktionen sind in unserem aktuellen API-Angebot nicht enthalten.
  </Tab>
</Tabs>
